# USD Beamer Theme
A professional LaTeX Beamer theme for the University of South Dakota, primed for math-heavy presentations using LaTeX.

## Features
* Official Colors
* Clean, Minimalist Layout
* Easily Configurable .sty File

## Installation
1. Clone repo or download `beamertheme-usd-theme.sty` and the `graphics/` folder.
2. Place them in the same directory as your .tex file.
3. Add `\documentclass[10pt,aspectratio=169]{beamer}` to your preamble along with `\usetheme{-usd-theme}`.
